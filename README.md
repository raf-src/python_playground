# python_playground

Technical sandbox for Python logic and user interactivity scripts.

- sorting_hat.py: Decision-based game using random chance and if/else logic.
- fairytale_adventure.py: Narrative generator using string formatting and user-input logic.
- magic_8_ball.py: Script providing randomized responses to user questions.

- Input handling / data types
- Conditional logic (if/elif/else)
- String formatting (f-strings)
- Standard library randomization
